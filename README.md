142.环形链表II 

快慢指针先找到交点，然后用数学公式算出从起点到循环点的长度与从交叉点到循环点的长度相等。然后用起点和交叉点找到循环点。
公式如下：
假设起点到循环点为x，循环点到交叉点为y，交叉点到循环点为z。
slow = 2 * (x + y) 因为fast走两步slow走一步\n
fast = x + y + n(y + z) n为fast循环次数\n
消除得：\n
x = (n - 1) (y + z) + z \n
设n为1，得x=z \n
